# MVP 开发路线图

## Phase 0：项目基建

- 单仓库初始化
- Docker 本地环境
- CI/CD 基础
- 环境变量治理

## Phase 1：账户与认证

- 手机号登录
- 实名认证提交
- 管理后台审核
- 基础权限体系

## Phase 2：订单交易闭环

- 用户发布订单
- 素材上传
- 整理师接单
- 用户确认锁单

## Phase 3：履约与计时

- 到达 GPS 校验
- 开工验证码
- 服务自动计时
- 订单完成确认

## Phase 4：支付与评价

- 预付款托管
- 结算账单
- 双向评价
- 信誉分扣减

## Phase 5：客服与维权

- 在线聊天
- 申诉与举报
- SOS 紧急求助
- 审计与证据保全

## Phase 6：运营后台

- 核心数据看板
- 财务统计
- 风控规则
- 城市/整理师排行

## 推荐节奏

1. 第 1-2 周：基建 + 账户认证
2. 第 3-5 周：订单交易主链路
3. 第 6-7 周：履约计时 + 支付
4. 第 8 周：评价、维权、后台闭环

## MVP 范围明确不做

- AI 图片分析
- AI 收纳用品推荐
- 团队接单
- 直播
- 会员体系
